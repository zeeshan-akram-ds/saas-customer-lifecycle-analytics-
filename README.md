# SaaS Customer Churn Analysis

This project analyzes customer churn behavior in a SaaS subscription business using real-world telecom dataset.

## Objective
Identify key factors driving customer churn and provide actionable insights to improve retention.

## Tools Used
- Python (Pandas, NumPy)
- Jupyter Notebook
- SQL (for analysis)
- Power BI (dashboarding - upcoming)

## Project Status
- Data loading and initial exploration completed
- Data cleaning and feature engineering in progress
